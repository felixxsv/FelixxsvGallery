When to use this

Use this page when you want to separate dedicated failure screens from lightweight errors that can stay in a toast.

Minor failure Retrying lists or fixing input issues can stay in existing toasts or modals.
Dedicated page Use this for initialization failures, access mismatches, or maintenance routing.
Dynamic display Customize details with ?code=, ?title=, ?message=, and ?request_id=.

Example: /gallery/error/?code=503&title=Temporarily Paused&request_id=abc123
