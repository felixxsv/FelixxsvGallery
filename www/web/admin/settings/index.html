Search and Filters

  • Use the header search to filter across titles, ALT text, tags, and users.
  • Use the sidebar to combine tags, color tags, shot dates, posted dates, and archives.
  • Archives are recalculated for the current filters and update when conditions change.

Viewing Images

  • Open a card to view the image in a larger modal.
  • Use the details action to inspect shot date, posted date, tags, and color tags.
  • Select the author icon to open the profile modal.

Account and Settings

  • Use the user icon in the top right to open profile editing, display settings, and account settings.
  • Email change, Discord link, two-factor auth, and logout are grouped in account settings.
  • Use /auth/ as the primary authentication flow.
