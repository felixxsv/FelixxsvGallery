What to check

Checking these items in order makes it easier to tell broken links from visibility restrictions.

Link Confirm the trailing URL and the path under / are correct.
Visibility Confirm the image or profile is not private or quarantined.
Retry Search again from the top page using search, tags, or archive.

Use this together with Apache or reverse-proxy 404 routing.
